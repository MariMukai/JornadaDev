# Parte 1 — Funções

Ex 1 — Função ou procedimento
Para cada item abaixo, responda se é uma função (retorna um valor) ou um procedimento (só
executa uma ação) e justifique em uma frase:

* Str() — converte número em string
* QOut() — exibe texto na tela
* Date() — retorna a data atual
* Len() — retorna o tamanho de uma string ou array
* Upper() — retorna a string em maiúsculas
* ClearScreen() — limpa a tela



Str() — converte número em string -> É uma function, pois converte e retorna uma string.

QOut() — exibe texto na tela -> É uma procedure, pois não há retornos.

Date() — retorna a data atual -> É uma function, pois retorna uma string de data atual.

Len() — retorna o tamanho de uma string ou array -> Function, pois retorna um inteiro INT.

Upper() — retorna a string em maiúsculas ->  Function, pois converte uma string para maiúscula e depois retorna uma nova string fotmatada.

ClearScreen() — limpa a tela -> Uma procedure, pois não tem retorno.
